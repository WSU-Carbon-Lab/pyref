Design Section 4 — Ingestion Pipeline + CatalogHandle

pub fn ingest_beamtime(
    beamtime_dir:    &Path,
    data_root:       Option<&Path>,  // NEW
    experimentalist: Option<&str>,   // NEW
    header_items:    &[String],
    force:           bool,
    progress_tx:     Option<Sender<(u32, u32)>>,

    // resolve catalog: DATA/.pyref/catalog.db
    // if data_root is None → falls back to old

        beamtime_dir, data_root

    let conn = open_catalog_db(&db_path)?;

    // upsert bt_beamtimes with new columns
    upsert_beamtime_record(&conn,
        beamtime_dir, data_root, experimentalist

    for (path, mtime) in paths {

            .map(|c| c.load(Ordering::Relaxed))
            .unwrap_or(false) { break; }
        …
    }
}
Backward compatibility: data_root: None and experimentalist: None reproduce the current behaviour exactly — the catalog is opened at the beamtime dir, no new columns are written. All existing callers (tests, Python bindings, CLI) pass None until they are individually migrated. The TUI always passes both.

Catalog Path Resolution

resolve_ingest_target()
pub fn resolve_ingest_target(
    beamtime_dir: &Path,
    data_root:    Option<&Path>,
) -> PathBuf {
    match data_root {
        // new path: DATA/.pyref/catalog.db
        Some(root) =>
            root.join(".pyref").join("catalog.db"),

        // legacy path: unchanged behaviour
        None =>
            resolve_catalog_path(beamtime_dir),
    }
}
Single write target. All beamtimes under the same DATA/ root share one catalog file. The bt_beamtimes.beamtime_path + source_path LIKE filter in queries is what isolates each beamtime's records.
upsert_beamtime_record()
// Called at the start of every ingest.
// Creates or updates the bt_beamtimes row.
fn upsert_beamtime_record(
    conn:            &Connection,
    beamtime_dir:    &Path,
    data_root:       Option<&Path>,
    experimentalist: Option<&str>,
) -> Result<()> {
    conn.execute("
        INSERT INTO bt_beamtimes
            (beamtime_path, data_root,
             experimentalist, last_indexed_at)
        VALUES (?1, ?2, ?3, strftime('%s','now'))
        ON CONFLICT(beamtime_path) DO UPDATE SET
            data_root        = excluded.data_root,
            experimentalist  = excluded.experimentalist,
            last_indexed_at  = excluded.last_indexed_at
    ", params![
        beamtime_dir.to_str(),
        data_root.and_then(|p| p.to_str()),
        experimentalist,
    ])?;
    Ok(())
}

CatalogHandle — Shared Query Surface

CatalogHandle struct
/// Lightweight handle — opens a fresh
/// connection per call (consistent with
/// existing rusqlite usage pattern).
pub struct CatalogHandle {
    pub db_path: PathBuf,  // DATA/.pyref/catalog.db
}

impl CatalogHandle {
    /// Explorer: list experimentalists + counts
    pub fn list_experimentalists(
        &self, data_root: &Path
    ) -> Result<Vec<ExptMeta>>;

    /// Explorer: list beamtimes for one expt
    pub fn list_beamtimes(
        &self,
        data_root:       &Path,
        experimentalist: &str,
    ) -> Result<Vec<BeamtimeMeta>>;

    /// Explorer: catalog status for one path
    pub fn catalog_status(
        &self, beamtime_path: &Path
    ) -> CatalogStatus;

    /// Beamtime screen: profiles query
    pub fn query_scan_points(
        &self,
        beamtime_path: &Path,
        filter:        Option<&CatalogFilter>,
    ) -> Result<Vec<FileRow>>;

    /// Beamtime screen: entries (samples/tags/scans)
    pub fn list_beamtime_entries_v2(
        &self, beamtime_path: &Path
    ) -> Result<BeamtimeEntries>;
}
ExptMeta + BeamtimeMeta return types
/// Returned by list_experimentalists()
/// — one row per depth-1 folder in catalog
pub struct ExptMeta {
    pub name:           String,
    pub beamtime_count: u32,
    pub fits_count:     u32,
    pub last_indexed:   Option<i64>,
}

/// Returned by list_beamtimes()
/// — one row per bt_beamtimes entry
pub struct BeamtimeMeta {
    pub path:           PathBuf,
    pub fits_count:     Option<u32>,
    pub last_indexed:   Option<i64>,
}

/// catalog_status() implementation:
/// 1. Query bt_beamtimes for beamtime_path
/// 2. If missing → NotIndexed
/// 3. If found → compare last_indexed_at
//     with max(mtime) of files in dir
/// 4. If dir mtime > last_indexed → Stale
/// 5. Otherwise → Indexed
Where CatalogHandle lives: Navigator owns one CatalogHandle constructed from data_root.join(".pyref/catalog.db"). Both ExplorerState and BeamtimeState call into it by reference — no cloning, no Arc. If the catalog file doesn't exist yet (first run, no ingests), all methods return empty vecs or NotIndexed; the DB is created on first ingest.
